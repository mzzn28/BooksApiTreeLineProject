api consumtion using Jquery for Books catalog

How to use:
1-clone the repository from https://github.com/mzzn28/BooksApiTreeLineProject
2-load the project in Visual studio
3-build and start the project.
4- once loaded it will be directed to default home page, to test result hit
	-'http://localhost:xxxx/api/Books' will show api result
	-'http://localhost:xxxxx/ComsumeApi.html' will display the result in tabular form.
